# MIMO Work

Public source bundle for MIMO Work.

This repository contains three independently prepared versions:

- `versions/current` - current MIMO Work desktop app source.
- `versions/macos-apple-silicon` - Apple Silicon macOS build workspace.
- `versions/windows` - Windows build workspace.

Generated build artifacts, dependency folders, logs, and local secret files are intentionally excluded from this repository.
