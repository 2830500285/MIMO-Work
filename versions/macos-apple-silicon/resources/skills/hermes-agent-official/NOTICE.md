# Hermes Agent Official Skills

This directory vendors the public `skills/` directory from the official Hermes Agent repository:

- Source: https://github.com/NousResearch/hermes-agent/tree/main/skills
- Commit: fcf6cb3d7304c644c3f52031598d0441ce1ff270
- Retrieved: 2026-06-18

Only public upstream repository content is included here. Local user skill directories such as
`~/.agents/skills`, `~/.codex/skills`, `~/.claude/skills`, `~/.hermes/skills`, and
`~/.mimo-work/skills` are intentionally not bundled into the application package.

The upstream license file is copied as `LICENSE.hermes-agent` in this directory.
